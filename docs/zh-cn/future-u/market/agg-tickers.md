# 获取全交易对的聚合行情信息

`GET` v1/future-u/market/public/q/agg-tickers


## 请求参数

注：此方法不需要签名

> 请求示例
```shell
curl --location --request GET 'https://api.ubitex.com/v1/future-u/market/public/q/agg-tickers' \
--header 'Accept: */*' \
--header 'Host: api.ubitex.com' \
--header 'Connection: keep-alive'
```
